# 3D Models und CAD-Dateien

Hier können 3D-Modelle für das PropagandaTruck-Fahrzeug abgelegt werden:

## Empfohlene Dateien:
- `chassis.stl` - Hauptchassis
- `camera_mount.stl` - Kamera-Halterung  
- `servo_brackets.stl` - Servo-Halterungen
- `electronics_case.stl` - Gehäuse für Elektronik
- `speaker_mount.stl` - Lautsprecher-Halterungen

## Formate:
- STL für 3D-Druck
- STEP/STP für CAD-Bearbeitung
- OBJ für Visualisierung
